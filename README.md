# 🎯 ciljaj brate

A 3D FPS aim training game inspired by Aim Lab and CS2 mechanics, built in Unity (URP).

## About

Ciljaj Brate is a first-person shooter training tool designed to improve aiming precision and game sense. It features two core modes — **Aim Training** and **Prefire Practice** — with an in-world settings panel where players change game settings by shooting at physical buttons on the map.

## Features

- 🎯 **Aim Training** — Multiple game modes for improving raw aim
- 🗺️ **Prefire** — CS2-inspired map routes for practicing common angles
- 🖱️ **Sensitivity System** — Fully adjustable sens, FOV and input smoothing
- 🌍 **In-World Settings** — Change modes and settings without leaving the game
- 📊 **Statistics** — Track accuracy, score and session performance

## Game Modes

| Mode | Description |
|------|-------------|
| Single Static | One target at a time, sequential spawns |
| Multi Static | Multiple targets on screen simultaneously |
| Speed Challenge | 30 targets — measured speed and accuracy |
| Timer Mode | How many targets in X seconds |
| Accuracy % | Tracks hit/total shot ratio |
| Moving Targets | Targets move in patterns *(coming soon)* |

## Visual Style

Aim Lab-inspired aesthetic with a custom color palette:
- **Map:** White / off-white geometry
- **Targets:** Dark pink / red (`#C0392B`, `#E91E8C`)
- **UI:** Minimal, same color palette

## Built With

- [Unity](https://unity.com/) — Universal Render Pipeline (URP)
